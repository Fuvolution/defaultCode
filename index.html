Purpose of this Document:

  • First, by creating a GitHub repository first and then cloning it onto the local machine shows the second (alternative) option for starting any even-numbered module challenge and any odd-numbered module challenge!
    1. For starting even-numbered module challenge, you can create a GitHub repo and then clone it to local machine without needing to do an additional git init, git pull origin main, and git remote add origin commands to get started!
    2. For starting odd-numbered module challenge, you need to create a GitHub repo to direct the origin attribute for git push origin commands. There are two scenarios:
      1. Clone the default code from someone else's GitHub repository onto the local machine -> Go to the cloned repository on the local machine -> Create a GitHub repo -> Copy new GitHub repo's HTTPS URL and do a git remote add origin (GitHub-repo-HTTPS-URL-inside-angle-brackets) -> git pull origin main on local machine terminal
      2. Create a GitHub repo -> Copy HTTPS URL of GitHub repo and git clone on local repo -> Go to the cloned repository on GitHub -> Copy HTTPS-URL of GitHub repo for default code and git clone onto the local repository -> git add -A -> git commit -m "Message" -> git push origin main
